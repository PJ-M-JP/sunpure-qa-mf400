# 组件安装机器人Q&A系统

## 项目说明
这是一个用于查询组件安装机器人相关问题与答案的智能查询系统，支持中英文切换、分类筛选、关键词搜索、新增Q&A等功能。

## 文件结构
- `index.html`：系统入口HTML文件，包含页面结构
- `style.css`：系统样式文件，定义页面视觉效果
- `data.js`：Q&A数据文件，包含所有问题与答案数据
- `script.js`：系统逻辑文件，实现交互功能
- `README.md`：项目说明文档

## 功能说明
1. **中英文双语切换**：支持中文/英文语言切换，所有Q&A内容均支持双语展示
2. **分类筛选**：支持按产品性能、应用场景、商务条款、公司情况、竞品相关进行分类筛选
3. **关键词搜索**：支持按关键词模糊搜索问题与答案内容
4. **新增Q&A**：支持新增Q&A内容，需要登录验证
5. **响应式设计**：适配电脑、平板、手机等多种设备
6. **卡片式展示**：Q&A内容采用卡片式折叠/展开展示，提升阅读体验

## 使用说明
1. 直接打开`index.html`即可使用系统
2. 分类筛选：点击顶部分类标签可筛选对应类别的Q&A内容
3. 搜索功能：在顶部搜索框输入关键词，可实时筛选相关Q&A内容
4. 语言切换：点击顶部语言切换按钮可切换中英文显示
5. 新增Q&A：点击顶部"新增Q&A"按钮，输入账号`PJ123`、密码`PJ123`登录后，可新增Q&A内容
